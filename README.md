# Itinerario di percorsi — ESMAS-ES+ — Italiano

## Descrizione

**Itinerario di percorsi - ESMAS-ES+ - Italiano** è una guida interattiva di un sistema di classificazione semantica. Questa risorsa organizza percorsi ontologici in diversi mondi concettuali, coprendo un'ampia varietà di argomenti. Ogni percorso è accompagnato da numerosi esempi lessicali per illustrarne il significato, rendendolo uno strumento completo per l'esplorazione del lessico.
